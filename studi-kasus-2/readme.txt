Tester on
	- Windows 10 pro
	- Xampp 5.6.12 (https://www.apachefriends.org/download.html)
		Disarankan menggunakan versi yang sama.
		Karena cara pengkodean PHP HTML masih menggunakan cara yang lama.
	- VSCode terbaru (https://code.visualstudio.com/download)
		Disarankan menggunakan text editor yang sama.
		Karena tampilan kode program ttp rapih.
	- PgSQL versi 13 (https://www.enterprisedb.com/downloads/postgres-postgresql-downloads)
	- DBeaver terbaru (https://dbeaver.io/download/)
	- Browser google chrome agar fitur2 HTML5 dapat berjalan baik
		
How to run.
	- Instal vscode sesuaikan apakah 32 bit atau 64 bit
	- instal xampp, lokasi "C:/xammp"
	- setelah xampp selesai di install, langsung tutup aplikasinya lewat taks-manager
	- run file xampp control-panel (klik kanan run administrator)
	- pastkan service apache diceklis dan runing diport 80:443
